#  DMart Sales Analysis — Power BI Dashboard

##  Project Overview

This project presents an interactive **Power BI dashboard** built to analyze **DMart retail store** sales data. The dashboard provides a comprehensive view of sales performance, customer behavior, order trends, and cancellation patterns using real-world retail data.

---

##  Objectives

- Analyze total sales, discounts, and order value across multiple dimensions
- Identify top-performing product categories, states, and ship modes
- Track monthly sales trends and seasonal patterns
- Understand customer behavior through subscription type and website engagement
- Monitor order status, payment status, and cancellation rates

---

##  Dashboard Visuals

| Visual Type | Description |
|-------------|-------------|
| KPI Cards | Total Sales, Total Discount, Average Rating |
| Funnel Chart | Total Sales by Ship Mode |
| Pie Chart | Order Value by State |
| Bar Chart | Order Value by Category |
| Clustered Column Chart | Order Value by Subscription Type |
| Line & Column Combo Chart | Total Sales by Month |
| Gauge Charts | Orders & Cancellation Rate |
| Clustered Bar Chart | Order Status & Payment Status |
| Word Cloud | Customer Insights |
| Bar Chart Race | Dynamic Sales Ranking |

---

##  Key Insights

- Identified **top-performing states and product categories** by total order value
- Tracked **monthly sales trends** to detect peak and low seasons
- Analyzed **ship mode preferences** and their impact on total sales
- Monitored **cancellation rates** vs total orders using gauge visuals
- Compared **payment status and order status** to understand fulfillment patterns
- Measured **customer engagement** via time spent on website and subscription behavior

---

##  Dataset Fields Used

- Total Sales, Total Discount, Discount Price
- Order Date, Month, Year
- Category, State, Ship Mode
- Subscription Type, Payment Status, Order Status
- Cancellation Date, Time Spent on Website
- Average Rating, Total Order Value
